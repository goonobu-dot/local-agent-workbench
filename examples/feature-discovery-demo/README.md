# Feature Discovery Demo

Fictional demo showing how Local Agent Workbench can turn a rough maintainer idea into a scoped feature decision.

The scenario is intentionally small: a maintainer wants to make first-run setup easier without building a hosted service or collecting telemetry.

Files:

- `question.md` - fictional feature brief
- `pane-roles.md` - pane assignments
- `pane-1-user-workflow.md` - user workflow notes
- `pane-3-cli-design.md` - CLI design notes
- `pane-4-tests.md` - acceptance and regression tests
- `decision-memo.md` - final scoped recommendation
- `handoff-summary.md` - handoff-style summary
