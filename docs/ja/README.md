# 日本語ドキュメント

Local Agent Workbench を日本語で導入・評価・紹介するための入口です。

## 目的別リンク

- まず動かす: [install.md](install.md)
- 使い始める: [quickstart.md](quickstart.md)
- 使いどころを理解する: [use-cases.md](use-cases.md)
- 不安を解消する: [faq.md](faq.md)
- SNSやプロフィールで紹介する: [share.md](share.md)
- OpenAIクレジット応募で説明する: [openai-credit-note.md](openai-credit-note.md)

## 推奨ルート

1. [install.md](install.md) の「試してから導入」で安全に確認する
2. [quickstart.md](quickstart.md) の Issue 調査ワークフローを動かす
3. [use-cases.md](use-cases.md) から自分の用途に近い型を選ぶ
4. 良い出力ができたら [share.md](share.md) の文例を調整して公開する
