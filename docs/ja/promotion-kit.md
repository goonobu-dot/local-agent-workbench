# 紹介キット

Local Agent Workbench を短く紹介したいときに使う文例です。

## 短い説明

Local Agent Workbench は、Codex CLI を最大9セッションまで1つの tmux ウィンドウに並べて使う無料のローカル作業台です。調査、OSSのIssue整理、PRレビュー、リリース準備、アイデア比較を、複数のAI視点で分担できます。

## 1行説明

Codex CLI を複数paneで動かし、調査・OSSメンテナンス・PRレビュー・リリース準備を並列化するローカル作業台。

## どんな人に向いているか

- Codex CLI をもっと使い倒したい人
- OSSのIssueやPRを整理したい人
- 1つのAIチャットだと考えが混ざってしまう人
- 調査やアイデア出しを複数視点で進めたい人
- Markdownとして作業ログや判断を残したい人

## 向いていない人

- Webダッシュボード型の完成品がほしい人
- AIに自動マージまで任せたい人
- 企業向けのSSOや集中管理が必要な人
