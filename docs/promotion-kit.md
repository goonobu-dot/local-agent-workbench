# Promotion Kit

Use this page when you want to explain Local Agent Workbench quickly.

## Short description

Local Agent Workbench is a free local workbench for running up to 9 Codex CLI sessions in one tmux window. It helps developers, researchers, and OSS maintainers split one problem into multiple AI-assisted angles and keep the output in one shared folder.

## One-line description

Run multiple Codex CLI panes locally for parallel research, OSS issue triage, PR review, release prep, and idea discovery.

## Who should try this

- People who already use Codex CLI and want a more organized workflow.
- OSS maintainers who want structured notes for issues, PRs, releases, and docs.
- Developers who like local-first tools and inspectable Markdown output.
- Researchers and builders who compare ideas from several angles.
- AI-agent enthusiasts who want a practical multi-pane setup without a hosted platform.

## Who should not use this

- People who want a polished web dashboard.
- Teams that need SSO, centralized policy controls, or enterprise audit systems.
- Users who expect AI output to be merged automatically without human review.
