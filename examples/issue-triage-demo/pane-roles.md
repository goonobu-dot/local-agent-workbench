# Pane Roles

Pane 1: Reproduce or restate the issue clearly.

Pane 2: Inspect likely code paths and identify where stdout/stderr may be mixed.

Pane 3: Propose a minimal fix plan.

Pane 4: Identify regression tests and edge cases.

Pane 5: Check security, privacy, or data-loss risk.

Pane 6: Draft the maintainer response.

Pane 7: Look for related issue patterns.

Pane 8: Challenge the leading diagnosis.

Pane 9: Synthesize the final triage memo.
