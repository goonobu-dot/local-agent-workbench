# 紹介文

## X投稿例

Codex CLIを最大9画面で並列に動かすローカル作業台を公開しました。

OSSのIssue調査、PRレビュー、リリース準備、機能アイデア出しを、1つの作業フォルダにまとめながら進められます。

導入チェック、デモ、公開前安全監査、サンプル出力も入れています。

https://github.com/goonobu-dot/local-agent-workbench

## 短い紹介文

Local Agent Workbench は、Codex CLI を複数paneで動かし、OSSメンテナンスや調査結果をMarkdownに集約するためのmacOS/tmux作業台です。

## GitHubプロフィール向け

AIエージェントをローカルで並列運用するためのワークベンチを開発しています。Issue調査、PRレビュー、リリース準備など、実際のOSSメンテナンスに使える形で公開しています。

## 日本語ユーザーに伝えるポイント

- まず `make demo` で試せる
- ローカルMacの中身がGitHubに公開されるわけではない
- ワークフロー共有前の安全検査がある
- 出力がMarkdownなので、Issue、PR、Obsidian、日報に転用しやすい
