# まずここから

このページは、初めて来た人が最初の3分で「これは何か」「次に何を読むか」を決めるための入口です。

## これは何か

Local Agent Workbench は、Codex CLI を複数画面で同じ作業フォルダに向けて動かすためのローカル作業台です。1つのAIチャットだけでは足りないときに、複数の視点で調査、整理、レビューを進められます。

## 最初の3分

1. まず [やさしい解説](BEGINNER_GUIDE.ja.md) を読みます。
2. 次に [ユースケース](USE_CASES.ja.md) から、自分の作業に近い使い方を選びます。
3. Macで試す前に `./scripts/doctor.sh` で必要なツールを確認します。
4. いきなり本番作業に使わず、まず `make demo` で動きを確認します。

## 読む順番

| あなたの状況 | 次に読むもの | 最初に試すこと |
| --- | --- | --- |
| 初めて見る | [やさしい解説](BEGINNER_GUIDE.ja.md) | `make demo` |
| OSSを整理したい | [ユースケース](USE_CASES.ja.md) | Issue整理 |

| 英語で読みたい | [Start Here](START_HERE.md) | [README](../README.md) |

## 安全面

このリポジトリを見るだけで、あなたのMacの中身が公開されることはありません。公開されるのは、自分でGitHubへ commit / push したファイルだけです。
