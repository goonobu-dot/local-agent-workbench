# Pane Roles

Pane 1: Draft changelog and compare it with the release scope.

Pane 2: Check breaking-change and compatibility risk.

Pane 3: Review README and docs readiness.

Pane 4: Build a smoke-test checklist.

Pane 5: Draft release notes.

Pane 6: Identify follow-up issues after release.

Pane 7: Challenge release readiness.

Pane 8: Check security and privacy wording.

Pane 9: Synthesize the final release handoff.
